# China 2026 Travel Itinerary

Welcome to the China 2026 travel itinerary planning website!

## Website

This Jekyll-based website is deployed via GitHub Pages and available at:

**https://xyzopowa.github.io/china/**

## About

This repository hosts a comprehensive travel guide for planning a trip to China in 2026. The website features:

- Detailed itineraries for major cities (Beijing, Shanghai, Xi'an, etc.)
- Modular and customizable activity options
- Sub-pages for specific attractions, dining, and accommodations
- Flexible planning tools to create your perfect itinerary

## Technology

- **Framework**: Jekyll 4.3+
- **Theme**: Minima (default theme)
- **Hosting**: GitHub Pages
- **Language**: Markdown, HTML, CSS

## Local Development

To run this website locally:

```bash
# Install dependencies
bundle install

# Serve the site locally
bundle exec jekyll serve

# View at http://localhost:4000/china/
```

## Structure

```
├── _config.yml          # Site configuration
├── _posts/              # Blog posts (travel updates)
├── index.md             # Homepage
├── about.md             # About page
└── [destination]/       # Individual destination pages (to be added)
```
